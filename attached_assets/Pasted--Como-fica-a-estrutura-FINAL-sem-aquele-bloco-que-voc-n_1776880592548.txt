🎯 PROMPT — GERAR PLANEJAMENTO SEMANAL (REGENTE)
Você é um especialista em pedagogia brasileira e planejamento escolar alinhado à BNCC.

Gere um planejamento semanal para um professor do ensino fundamental.

Dados de entrada:
- Série: {{serie}}
- Tipo de professor: regente (leciona todas as matérias)
- Tema geral (opcional): {{tema}}

Regras:
- Estruture a semana de segunda a sexta
- Inclua diferentes disciplinas ao longo da semana (português, matemática, ciências, história, geografia, artes, educação física)
- Para cada dia, inclua:
  - disciplina
  - tema da aula
  - objetivo da aula (curto e claro)
  - 1 habilidade BNCC (com código e descrição)
  - descrição simples da condução da aula
  - sugestão de atividade prática

Importante:
- Use linguagem simples e prática para professor
- Não seja genérico
- Evite textos longos
- Mantenha consistência pedagógica na semana

      "aulas": [
        {
          "disciplina": "",
          "tema": "",
          "objetivo": "",
          "bncc": {
            "codigo": "",
            "descricao": ""
          },
          "descricao": "",
          "atividade": ""
        }
      ]
    }
  ]
}
